## 개인과제 필수문제 2

<details>
  <summary>과제 상세보기</summary>

**쇼핑몰 상품 가격을 기반으로 장바구니 상태를 확인하고,  
총 금액이 2만원 이상일 경우 10% 할인을 적용하는 프로그램입니다.**

### ✔ 과제 설명
- Map 자료형으로 상품명과 가격을 저장  
- List 자료형으로 장바구니를 구성  
- 장바구니 개수에 따라 다른 메시지 출력  
- for-in 반복문으로 장바구니 총 금액 계산  
- 총합이 **20,000원 이상이면 10% 할인 적용**  
- 할인 금액과 최종 결제 금액을 출력  
- 총합이 2만원 미만일 경우 정상 금액 출력  
- 장바구니가 완전히 비어 있을 경우 안내 메시지 출력  

### ✔ 구현 내용
- `Map<String, int>` : 상품명 → 가격 저장  
- `List<String>` : 장바구니 아이템  
- `.length` : 장바구니의 총 아이템 개수  
- `for(String item in cart)` : 장바구니 돌며 가격 합산  
- `if / else` :  
  - 장바구니 상태 출력  
  - 할인 조건 체크  
  - 최종 결제 금액 출력  
- `toInt()` : 할인된 금액 소수점 제거  

</details>

## 개인과제 도전문제 1

<details>
  <summary>과제 상세보기</summary>

**1~45 사이의 숫자에서 중복 없이 6개를 뽑아 로또 번호를 생성하고,  
발급된 번호를 당첨 번호와 비교하여 등수를 판정하는 프로그램입니다.**

### ✔ 과제 설명
- 1~45 범위의 난수를 생성하여 번호 6개 만들기  
- Set 자료형으로 **중복 제거**  
- while 반복문으로 번호가 6개가 될 때까지 계속 뽑기  
- 번호를 List로 변환하여 정렬  
- 내가 뽑은 번호와 당첨 번호 비교  
- for-in 반복문 + contains()로 **일치 개수 계산**  
- 일치 개수에 따라 등수 출력  
- clear()로 로또 번호 초기화  

### ✔ 구현 내용
- `Random().nextInt(45)+1` : 1~45 난수 생성  
- `Set<int>` : 중복 없는 로또 번호 저장  
- `List<int>` : 정렬 가능한 리스트로 변환  
- `for(int n in myLotto)` : 번호 하나씩 꺼내며 비교  
- `winNums.contains(n)` : 당첨 번호 포함 여부 체크  
- `if / else if / else` : 등수 판단  
- `.clear()` : 리스트 초기화  

### ✔ 어려웠던 점
- **함수 구조가 헷갈림**  
  - 함수 안에서 `return` 위치나 중괄호 `{}` 배치가 잘 기억 안 나서 몇 번을 다시 쓰고 지우기를 반복했음.  
  - 특히 `void main()` 밖에 함수를 두는 구조가 처음엔 익숙하지 않아 계속 틀리게 작성함.
- **자료형(List, Map) 다룰 때 혼란**  
  - 장바구니는 `List`이고 상품은 `Map`인데, 두 개를 섞어 사용할 때 `key`인지 `value`인지 순간적으로 헷갈림.  
  - `contains()` 사용 시 `"문자"`를 찾는 건지 숫자를 찾는 건지 한 번에 떠오르지 않아 여러 번 수정함.
- **for-in 반복문 로직이 머릿속에 잘 안 잡힘**  
  - “아이템 하나씩 꺼내서 계산한다”는 개념은 알겠는데, 실제 코드로 옮길 때 매번 구조가 틀어져서 반복적으로 다시 작성해야 했음.
- **조건문 분기 처리에서 오타와 괄호 실수 지속 발생**  
  - `if / else` 구조는 이해했지만, 조건문 끝에 `;` 찍거나 괄호가 어긋나는 실수가 반복됨.  
  - 조금만 집중력이 흐트러져도 문법오류가 나와서 멘탈이 흔들렸음.
- **할인 로직 구현 시 순서 헷갈림**  
  - “총합 계산 → 조건 체크 → 할인 적용 → 소수점 제거” 흐름은 머릿속에 있는데, 코드에서는 순서가 뒤죽박죽 되어 다시 배치해야 했음.
- **계속 반복해서 다시 작성해야 하는 스트레스**  
  - 여러 번 같은 코드를 다시 쓰는 과정에서 “왜 난 이것도 못하지?”라는 조급함이 강하게 느껴졌음.  
  - 하지만 다시 해보면 결국 다 성공했기 때문에, 반복 연습이 효과가 있다는 걸 스스로도 확인함.
  
</details>

## 개인과제 필수문제 New

<details>
  <summary>과제 상세보기</summary>

**학생들의 이름과 점수를 파일에서 불러와, 특정 학생의 평균 점수를 계산하고
출력한 결과를 파일에 저장하는 "학생 점수 분석 프로그램"**

### ✔ 과제 설명

- `students.txt` 파일에서 `"이름,점수"` 형식의 데이터를 읽기  
- Score / StudentScore 클래스를 이용한 **객체지향 구조 설계**
- `loadData()` : 파일 읽고 → 오류 검사 → 객체 생성 → 리스트 저장  
- `getResult()` : 사용자 입력 → 해당 학생 평균 계산  
- `saveResult()` : 평균 결과를 `result.txt`에 저장  
- main()에서 전체 흐름 통합  

---

### ✔ 구현 내용

- **클래스 구조**
  - `Score` : 점수만 보유한 부모 클래스  
  - `StudentScore extends Score` : 이름 + 점수, @override  

- **파일 처리**
  - `File(path).readAsLinesSync()` 로 줄 단위 읽기  
  - 공백/형식 오류/점수 오류 시 각각 스킵  
  - 정상 데이터만 StudentScore 객체로 변환 후 리스트에 추가  

- **평균 계산**
  - for-in 반복문으로 이름 일치 항목만 sum / eo  
  - `sum ~/ eo` 정수 나눗셈으로 평균 계산  

- **입력 검증**
  - `stdin.readLineSync()` 로 입력  
  - null / 공백 입력 시 재입력 요청  

- **결과 저장**
  - `writeAsStringSync()`로 파일 저장
