## 개인과제 필수문제 1

<details>
  <summary>과제 상세보기</summary>

**점수(score)에 따라 A, B, C 등급을 출력하는 조건문 연습 과제입니다.**

### ✔ 과제 설명
- `int score` 값을 기준으로 등급 판정  
- 조건문 `if / else if / else` 사용  
- 점수 범위

  - 80~89점 → B등급  
  - 79점 이하 → C등급

- 정수형 변수 `score` 선언  
- 조건문 분기 처리  
- 논리 연산자 `&&` 사용  
- 점수 구간별 다른 출력

## 개인과제 필수문제 2

<details>
  <summary>과제 상세보기</summary>

**쇼핑몰 상품 가격을 기반으로 장바구니 상태를 확인하고,  
총 금액이 2만원 이상일 경우 10% 할인을 적용하는 프로그램입니다.**

### ✔ 과제 설명
- Map 자료형으로 상품명과 가격을 저장  
- List 자료형으로 장바구니를 구성  
- 장바구니 개수에 따라 다른 메시지 출력  
- for-in 반복문으로 장바구니 총 금액 계산  
- 총합이 **20,000원 이상이면 10% 할인 적용**  
- 할인 금액과 최종 결제 금액을 출력  
- 총합이 2만원 미만일 경우 정상 금액 출력  
- 장바구니가 완전히 비어 있을 경우 안내 메시지 출력  

### ✔ 구현 내용
- `Map<String, int>` : 상품명 → 가격 저장  
- `List<String>` : 장바구니 아이템  
- `.length` : 장바구니의 총 아이템 개수  
- `for(String item in cart)` : 장바구니 돌며 가격 합산  
- `if / else` :  
  - 장바구니 상태 출력  
  - 할인 조건 체크  
  - 최종 결제 금액 출력  
- `toInt()` : 할인된 금액 소수점 제거  

</details>
